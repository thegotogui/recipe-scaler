Density References

This is a reference for densities of common ingredients.

When you set an Ingredient’s density, you can do calculations using any mass unit or volume unit.

About Density

Density is the amount of mass per volume for an Ingredient.

Enter Density and Fillet can do conversions between mass units and volume units wherever this Ingredient is used.

If an Ingredient does not have Density set, Recipes or Menu Items using that Ingredient will have calculation errors.

If you often switch between mass and volume measurements, it’s a good idea to set up density for your key ingredients.
